## 概要

ホーム画面の「検査処理設定」から遷移するプロジェクト管理画面。検査フロー・AIモデルをまとめる単位である「プロジェクト」の一覧表示、新規作成、削除を行う。ユーザーはプロジェクトカードをダブルクリックしてフローエディタ（SCR-005）へ進む。MVP時点でのプロジェクト作成ダイアログはプロジェクト名の入力のみ。Phase 4以降でテンプレート選択UIがダイアログ内に追加される。
