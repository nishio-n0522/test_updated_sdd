## サイドバー（グローバルナビゲーション）

### 概要

全画面で共通のサイドバー。画面左端に配置され、アプリ内の主要機能エリアへのナビゲーションを提供する。最上部にアプリロゴを配置し、その下に各機能のナビ項目を縦に並べて表示する。

### 表示状態

| 状態     | 対象画面        | 説明                                                                                                 |
| -------- | --------------- | ---------------------------------------------------------------------------------------------------- |
| 閉状態   | SCR-001         | アイコンのみ表示（細幅）。ホバーでオーバーレイ展開する。コンテンツ領域に重なる形で展開し、マウスが離れると閉状態に戻る |
| 開状態   | SCR-002〜SCR-009 | アイコン＋ラベル常時表示。固定幅のナビゲーション領域として常に表示される                              |

### 共通操作挙動

#### 操作: 非活性ナビ項目をクリック

**トリガー**: ユーザーがサイドバーのグレーアウト状態のナビ項目（画像管理・学習管理・検査処理分析・検査稼働・クラウド）をクリックする
**前提条件**: クリックした項目が非活性状態（対応フェーズ未到達）

**正常系の挙動**:

1. クリックされた項目の近傍に「この機能は今後対応予定です」というツールチップが表示される
2. ツールチップは約2秒後に自動的に消える
3. 画面遷移は発生しない

**異常系の挙動**:

| 条件         | 挙動             |
| ------------ | ---------------- |
| 該当なし     | -                |

**画面遷移**: 遷移なし
**状態変化**: 変化なし（ツールチップ表示のみ）

> この操作は全画面（SCR-001〜SCR-009）で共通の挙動である。SCR-001ではサイドバー展開時およびコンテンツエリアのグリッドボタンの両方で同一の挙動が適用される。
