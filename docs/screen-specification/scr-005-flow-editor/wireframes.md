# SCR-005: フローエディタ画面

> 仕様ファイル: `docs/screen-specification/scr-005-flow-editor/wireframes.md`

## 概要

検査パイプラインをノードベースで視覚的に設計するメイン画面。SCR-004（プロジェクト一覧）から既存プロジェクトを開くか新規プロジェクト作成後に遷移する。Node-RED風のUIを採用し、左端のグローバルナビゲーションサイドバー（開状態）・ノードパレット・中央キャンバス・右端のプロパティパネル（ノード選択時）の4エリアで構成される。フロー設定は30秒以内に自動保存される。
